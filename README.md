# Retail Executive Sales Performance Dashboard

## Business Problem Summary

The retail leadership team requires an executive dashboard to monitor overall business performance across sales, profitability, customer segments, product categories, shipping performance, discount impact, and return patterns. The objective is to identify trends, risks, and opportunities that support better business decision-making.

---

## Dataset Description

The dataset contains transactional retail sales information including:

- Order Date
- Region
- Customer Segment
- Category
- Sub-Category
- Sales
- Profit
- Discount
- Delivery Days
- Return Flag
- Customer Rating
- Ship Mode

The data was imported into Tableau Public for dashboard development.

---

## Tableau Workbook Description

The Tableau workbook contains an executive dashboard built for business leaders.

The dashboard includes:

- Monthly Sales Trend
- Regional Sales & Profit Performance
- Category & Sub-Category Profitability
- Customer Segment Performance
- Shipping Performance
- Discount vs Profit Analysis
- Return Analysis

It also includes KPI cards for:

- Total Sales
- Total Profit
- Average Customer Rating

---

## Calculated Fields Created

The following calculated fields were created:

- Profit Margin
- Cost
- Average Order Value
- Return Rate
- Shipping Delay Bucket

---

## Dashboard Components

The dashboard contains:

- 7 business visualizations
- 3 KPI cards
- Interactive filters
- Business-friendly layout
- Executive summary view

---

## Filters and Interactions Used

Interactive filters include:

- Region
- Category
- Customer Segment

Selecting filters updates all dashboard visualizations.

---

## Key Business Insights

- Sales fluctuate throughout the year.
- South region generates the highest profit.
- Technology products generate the highest profitability.
- Home Office customers contribute the highest sales.
- Standard shipping has the longest delivery time.
- Higher discounts generally reduce profitability.
- Furniture experiences the highest number of returns.

---

## Dashboard Story Summary

The dashboard provides executives with a single view of business performance by combining sales trends, profitability, customer behavior, shipping efficiency, and return analysis to support strategic decision-making.

---

## Assumptions and Limitations

- Dashboard results depend on the available dataset.
- Return Flag is treated as a binary variable.
- Customer Rating values are assumed to be complete.
- External business factors are not included.
